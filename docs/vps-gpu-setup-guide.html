Tài liệu Setup

VieNeu-TTS trên VPS GPU

Hướng dẫn từng bước cài đặt và cấu hình hệ thống tổng hợp giọng nói VieNeu-TTS (hỗ trợ chuyển đổi SRT sang Audio đồng bộ hóa timeline) trên máy chủ ảo GPU thuê. Chỉ cần copy và paste lệnh.

GPU RTX 3090 Base pytorch/pytorch:latest OS Linux (Ubuntu) Port 8999 (Exposed)
BƯỚC 0

Mở cửa sổ dòng lệnh (Terminal)

Trên giao diện điều khiển JupyterLab của máy chủ ảo GPU (ví dụ: Vast.ai, RunPod, ...), hãy nhìn xuống mục "Other" ở phía dưới và click vào nút Terminal. Một cửa sổ dòng lệnh nền đen sẽ mở ra.

Tất cả các dòng lệnh trong hướng dẫn này sẽ được thực thi trực tiếp tại cửa sổ Terminal này. Hãy copy chính xác và paste để chạy.
BƯỚC 1

Kiểm tra tình trạng hoạt động của GPU

Đảm bảo máy chủ ảo của bạn đã nhận diện đầy đủ phần cứng card đồ họa (GPU) và driver CUDA hoạt động bình thường.

terminal 
nvidia-smi
Nếu màn hình hiển thị tên dòng GPU (ví dụ: RTX 3090, RTX 4090) cùng với phiên bản CUDA đi kèm là hệ thống đã sẵn sàng.
BƯỚC 2

Cài đặt Build Tools & Trình quản lý gói UV

Cài đặt các thư viện hệ thống cần thiết để biên dịch (build) mã nguồn Python và công cụ quản lý gói siêu tốc uv. Hãy sao chép và chạy lần lượt từng dòng lệnh dưới đây.

1. Cài đặt các công cụ biên dịch 
apt-get update && apt-get install -y build-essential cmake
2. Xóa cấu hình CC cũ (Tránh lỗi build llama-cpp) 
unset CC CXX
3. Cài đặt trình quản lý gói UV 
curl -LsSf https://astral.sh/uv/install.sh | sh && source ~/.bashrc
BƯỚC 3

Tải mã nguồn (Clone Repo) và cài đặt thư viện

Truy cập vào thư mục làm việc, tải mã nguồn VieNeu-TTS về và tự động cài đặt tất cả các gói phụ thuộc hỗ trợ tăng tốc trên GPU đồ họa.

1. Clone kho lưu trữ 
cd /workspace && git clone https://github.com/pnnbao97/VieNeu-TTS.git voice_tts && cd voice_tts
Nếu bạn đang sử dụng kho lưu trữ (repository) riêng tư có mã nguồn xử lý SRT tùy biến, hãy thay liên kết trên thành liên kết repo của bạn:
git clone https://github.com/TÊN_USER/TÊN_REPO.git voice_tts
2. Đồng bộ hóa gói phụ thuộc GPU qua UV 
uv sync --group gpu

Quá trình cài đặt thư viện này có thể mất từ 5 đến 15 phút tùy thuộc vào hiệu năng mạng của máy chủ. Hãy kiên nhẫn đợi cho đến khi Terminal thông báo dòng chữ Installed ... packages.

BƯỚC 4

Khởi chạy VieNeu-TTS (Tối ưu hóa RTX 3090)

Thiết lập biến môi trường và chạy dịch vụ web tổng hợp giọng nói. Lệnh này đã được cấu hình tối đa hóa hiệu năng cho card RTX 3090 (24GB VRAM).

Chạy máy chủ (Copy toàn bộ khối này) 
GRADIO_SERVER_NAME=0.0.0.0 \
GRADIO_SERVER_PORT=8999 \
LMDEPLOY_MEMORY_UTIL=0.8 \
LMDEPLOY_PREFIX_CACHING=1 \
LMDEPLOY_MAX_BATCH_SIZE=8 \
uv run vieneu-web
Biến cấu hình Giá trị Ý nghĩa chi tiết
MEMORY_UTIL 0.8 Sử dụng 80% bộ nhớ GPU (~19.2GB VRAM) làm cache KV giúp tăng tốc đáng kể tốc độ suy luận.
PREFIX_CACHING 1 Kích hoạt bộ nhớ đệm tiền tố (Prefix Caching) cho giọng nói mẫu, tăng tốc khi tái sử dụng cùng giọng nói.
MAX_BATCH_SIZE 8 Xử lý song song tối đa 8 yêu cầu đồng thời (cực kỳ hữu ích khi tổng hợp file phụ đề SRT chứa nhiều câu ngắn).
SERVER_PORT 8999 Cổng dịch vụ đã được mở (expose) của VPS để truy cập ra mạng Internet bên ngoài.

Khi quá trình tải mô hình và tối ưu hóa kết thúc thành công, bạn sẽ thấy Terminal in ra dòng chữ:

Dòng trạng thái thành công mong đợi 
* Running on local URL:  http://0.0.0.0:8999
   ✅ Optimized backend initialized
BƯỚC 5

Mở trang web giao diện và sử dụng

Sau khi máy chủ khởi chạy thành công, hãy mở một thẻ trình duyệt web mới và truy cập vào đường dẫn giao diện bên dưới:

Đường dẫn truy cập web (tự động cập nhật theo IP nhập bên trái) 
http://209.121.195.118:12018

Giao diện điều khiển trực quan Gradio sẽ hiện ra với các thẻ chức năng chính:

Thẻ chức năng (Tab) Mô tả hoạt động chi tiết
Text to Speech Nhập đoạn văn bản tùy ý tiếng Việt/tiếng Anh, chọn giọng mẫu sẵn có hoặc tùy chỉnh để tạo âm thanh tức thì.
Voice Cloning Tải lên một file âm thanh mẫu ngắn (3 đến 5 giây) để nhân bản giọng đọc đó và lưu lại sử dụng.
SRT → Audio Dán nội dung file phụ đề SRT vào để hệ thống tự động sinh các file âm thanh khớp thời gian hiển thị phụ đề.
Podcast Thiết lập hội thoại nói chuyện/phỏng vấn qua lại giữa nhiều nhân vật có giọng nói khác nhau.
HƯỚNG DẪN LỖI

Khắc phục khi gặp sự cố hoặc lỗi phát sinh

Nếu trong quá trình vận hành xảy ra lỗi, bạn có thể nhanh chóng xử lý theo các bước sau:

Lỗi hết bộ nhớ đồ họa "Out of Memory" (OOM)

Trường hợp xảy ra khi chạy các mô hình lớn hoặc trên cấu hình GPU VRAM thấp hơn 24GB. Hãy hạ tỷ lệ chiếm dụng VRAM (LMDEPLOY_MEMORY_UTIL) xuống 0.5 và giảm batch size bằng lệnh sau:

terminal — chạy lại với VRAM thấp hơn 
GRADIO_SERVER_NAME=0.0.0.0 \
GRADIO_SERVER_PORT=8999 \
LMDEPLOY_MEMORY_UTIL=0.5 \
LMDEPLOY_PREFIX_CACHING=1 \
LMDEPLOY_MAX_BATCH_SIZE=4 \
uv run vieneu-web
Lỗi cổng mạng đang bị chiếm dụng "Port already in use"

Xảy ra khi tiến trình của VieNeu-TTS chạy ngầm hoặc chưa được giải phóng hoàn toàn từ phiên làm việc trước. Hãy sử dụng lệnh sau để cưỡng chế dừng toàn bộ tiến trình VieNeu-TTS đang chạy:

terminal — tắt toàn bộ tiến trình vieneu-web 
pkill -9 -f vieneu-web

Hoặc giải phóng trực tiếp bằng cách dọn cổng 8999:

terminal — giải phóng cổng 8999 
fuser -k 8999/tcp
Lỗi build gói "llama-cpp-python"

Nếu quá trình uv sync gặp lỗi build, hãy đảm bảo bạn đã chạy hai lệnh chuẩn bị này trước để cài đặt gói cmake cần thiết và reset biến môi trường biên dịch cũ:

terminal — sửa lỗi build 
apt-get update && apt-get install -y build-essential cmake
unset CC CXX
Một số phím tắt cơ bản
  • Muốn dừng server đang chạy: Nhấn tổ hợp phím Ctrl + C ngay tại cửa sổ Terminal đang chạy máy chủ.
  • Muốn khởi chạy lại sau khi đã dừng: Bạn chỉ cần chuyển hướng vào đúng thư mục dự án và thực hiện lại bước chạy máy chủ (Bước 4).
TÓM TẮT NHANH

Lệnh chạy nhanh (áp dụng cho những lần sau)

Khi bạn đã hoàn thành cài đặt toàn bộ hệ thống ở các bước trên, ở những lần tiếp theo khi khởi động lại VPS, bạn không cần phải làm lại từ đầu. Chỉ cần mở Terminal lên và thực hiện một dòng lệnh gộp duy nhất bên dưới:

terminal — khởi chạy nhanh 
cd /workspace/voice_tts && GRADIO_SERVER_NAME=0.0.0.0 GRADIO_SERVER_PORT=8999 LMDEPLOY_MEMORY_UTIL=0.8 LMDEPLOY_PREFIX_CACHING=1 LMDEPLOY_MAX_BATCH_SIZE=8 uv run vieneu-web
